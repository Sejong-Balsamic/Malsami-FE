## 📝 현재 문제점

대규모 레거시 제거·API 통일·스타일 통일·구조 정리 작업 이후, 회귀(regression) 없이 주요 사용자 플로우가 정상 동작하는지 검증이 필요합니다.

- 백엔드 로컬 서버는 미기동 상태이나, 프론트 `appClient`의 baseURL이 운영 API(`https://api.sejong-malsami.co.kr`)로 설정되어 있어 운영 API 기준 검증이 가능합니다.
